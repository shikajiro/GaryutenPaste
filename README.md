# GaryutenPaste
This SublimeText plug-in paste erase the newline at the end of the text in the clipboard.

## directory
├Sublime Text 2
  ├─Packages
    ├─GaryutenPaste
      ├── Default\ (Linux).sublime-keymap
      ├── Default\ (OSX).sublime-keymap
      ├── Default\ (Windows).sublime-keymap
      ├── GaryutenPaste.py
      └── README.md

## sample
"abcde\n"
↓
"abcde"

## how to use
command + v
